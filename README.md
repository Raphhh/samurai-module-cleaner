Files cleaner module for [Samurai](https://github.com/Raphhh/samurai).

Remove files from your bootstrap to clean your project. 

Removed files:
 - `CHANGELOG`
 - `UPGRADE`
 - `CONTRIBUTING`

## Installation

```console
$ samurai module install cleaner raphhh/samurai-module-cleaner
```
